## Website Prompt Generator (Lovable + Gemini)

This is a small bilingual (English/Arabic) web app that helps users generate a clear, ready-to-paste prompt for creating websites in **Lovable**.

Users:
- choose one of 5 domains (healthcare, smart cities, business, education, non-profit / volunteering),
- answer a few short questions about their idea,
- and get a single, well-structured prompt generated by **Gemini**.

### Tech stack

- Static frontend: `index.html` + `styles.css` + `app.js`
- Backend: a single Vercel serverless function `api/generatePrompt.js` that calls the Gemini API

No framework is required.

### Local development

You can open `index.html` directly in the browser for UI-only testing.  
The "Generate prompt" button will work correctly only when the `/api/generatePrompt` endpoint is available (for example when running on Vercel or a similar environment).

### Deploying to Vercel

1. **Push this folder to a Git repository** (GitHub, GitLab, etc.).
2. **Create a new Vercel project** and import that repository.
3. In the project settings on Vercel, add the following environment variable:

   - **Key**: `GEMINI_API_KEY`  
   - **Value**: your Gemini API key

4. Deploy the project. Vercel will:

   - serve `index.html` as the main page,
   - expose the serverless function at `/api/generatePrompt`.

### How the Gemini integration works

- The frontend (`app.js`) sends a `POST` request to `/api/generatePrompt` with:

  ```json
  {
    "language": "en" | "ar",
    "domain": "healthcare" | "smartCities" | "business" | "education" | "nonProfit",
    "answers": {
      "...": "..."
    }
  }
  ```

- `api/generatePrompt.js`:
  - builds a system-style prompt using the domain and answers,
  - calls `gemini-1.5-pro-latest` via the public HTTP API,
  - returns `{ "prompt": "<final prompt text>" }`.

The app shows that text in a textarea with a "Copy to clipboard" button so the user can paste it directly into Lovable.
